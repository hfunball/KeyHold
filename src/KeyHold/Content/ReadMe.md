# KeyHold

## Things to know

- KeyHold is for games and simple Windows tasks where holding movement keys gets repetitive.
- Hold the keys you want KeyHold to take over, then press the toggle trigger. The default trigger is Home.
- The toggle trigger can be a keyboard key, Mouse Button 4, Mouse Button 5, or Middle Mouse.
- Press the toggle trigger again to stop. In some games, pressing and releasing the held keys is the most reliable way to take control back.
- KeyHold may not behave like a normal text tool in Notepad, office apps, chat apps, or browsers. Games and Windows apps look at keyboard input in different ways.
