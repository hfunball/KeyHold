# Privacy

RunHold runs locally on your Windows PC.

It does not collect, transmit, sell, or upload data. RunHold has no networking code and no telemetry.

RunHold monitors keyboard state and supported mouse trigger buttons so it can detect the configured toggle binding. It stores only app settings, such as the configured trigger, theme, startup preference, and notification preference. It does not store raw typed text, passwords, chat messages, or game input history.

Settings are stored in:

```text
%LOCALAPPDATA%\RunHold\settings.json
```
